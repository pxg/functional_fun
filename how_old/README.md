# How Old Am I?
You will need the Haskell platform installed <https://www.haskell.org/platform/>.

## V1
To compile:
```
ghc -o how_old_am_i how_old_am_i.hs
```

To run:
```
./how_old_am_i
```

## V2
To compile and run:
```
ghc -o How_old Main.hs
./How_old
```

Run the tests:
```
runghc Tests.hs
```
